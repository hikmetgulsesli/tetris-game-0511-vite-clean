# Tetris Game Vite Clean PRD ## Overview Tetris Game Vite Clean turns the user's request into a directly usable application. The first screen is not a landing page; it is the real workflow surface where the user can inspect data, take actions, and recover from empty or error states. User-facing copy language: English. ## Goals - Provide a working first-load experience. - Ensure every visible button triggers a real behavior. - Use Lucide React for icons. - Persistent state via localStorage. ## Screens 1. Playfield: Playable main scene, score, next piece, and primary controls. 2. Main Menu: Start game, difficulty, settings, and resume actions. 3. Results: Game over outcome, high score, replay, and menu actions. 4. Settings: Audio, difficulty, and control preferences. 5. Help: Short rules plus keyboard and touch explanations. ## Aesthetic Dark, modern, corporate/minimal. Primary #2563EB, Background #0F172A.
